## Legenda

- **Volume**: estimativa mensal Brasil (M=Mensal). `~` = estimado; intervalos quando incerteza alta.
- **KD**: dificuldade de ranqueamento orgânico (1-100). >70 = competitivo (Conviso/Tempest/Tivit dominando).
- **Intenção**: T=Transacional (BOFU) / S=Solution (MOFU) / I=Informacional (TOFU)
- **Match**: match type recomendado para Google Ads ([exact], "phrase", broad)
- **Destino**: página atual ou recomendação de criação
